    metadata let's the entities and the browser know
    what the webpage is about and what it contains.


    <meta charset="UTF-8">
    - Unicode characters
    
    <meta name="keywords" content="HTML,meta tags,meta,learning">
    - Keywords of the website (like youtube tags)
    
    <meta name="description" content="Learning meta tags in HTML.">
    - Description of the website on search tab
    
    <meta name="author" content="Ario">
    - Author of the website

    <meta name="viewport" content="width=device-width,initial-scale=1.0">
    - The webpage size scaling (since devices have different screen sizes)

    <meta http-equiv="refresh" content="3"> 
    - Auto refresh every 3 seconds
